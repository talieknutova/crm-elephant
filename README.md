# 🐘 Elephant Care CRM

Проект представляет собой веб-приложение для управления заповедником слонов. Приложение решает бизнес-задачу учета животных, автоматического подсчета ежедневной потребности в провизии и контроля за состоянием здоровья слонов.

# Основной функционал

* 🐘 Умный склад еды: Раздельный учет сена/травы и веток. Запасы убывают каждый "день" автоматически.

* 🐘 Биологическая модель: - Динамический возраст (рассчитывается от системной даты и даты рождения).

* 🐘 Потребление пищи рассчитывается по сложной формуле (зависит от возраста, веса и рода слона).

* 🐘 Учет таксономии (Африканские слоны едят больше веток, Азиатские — больше травы).

* 🐘 Машина времени: Переключение дней и возможность "отматывать" время назад или вперед.

* 🐘 Ретроактивность: При добавлении слона "задним числом" система сама пересчитает и удержит со склада еду за пропущенные дни.

* 🐘 Уход за животными: Слоны пачкаются, их нужно мыть 🚿. Могут болеть 💊. При необходимости слона можно проводить в "слоновий рай" ☁️.

* 🐘 Журналирование: Каждое действие записывается в специальный лог с привязкой к дате.


# 🐘 Технологии

* 🐘 Backend: Python, Flask (PEP-8 compliant)

* 🐘 Database: SQLite (через Flask-SQLAlchemy)

* 🐘 Frontend: HTML, TailwindCSS (интерактивные POST-запросы без перезагрузки)

# 🐘 Как запустить проект (Инструкция)

1. 🐘 Склонируйте репозиторий:
```
git clone <url_вашего_репозитория>
cd elephant-crm
```

2. 🐘 Создайте и активируйте виртуальное окружение:
```
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для Mac/Linux:
source venv/bin/activate
```

3. 🐘 Установите зависимости:
```
pip install -r requirements.txt
```

4. 🐘 Инициализируйте 🐘 базу 🐘 данных (создаст 🐘 тестовый 🐘 набор 🐘 слонов🐘🐘🐘):
```
python init_db.py
```

5. 🐘 Запустите сервер:
```
python app.py
```

6. 🐘 Откройте браузер: Перейдите по адресу http://127.0.0.1:5000
